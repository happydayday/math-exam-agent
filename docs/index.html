MathExam AI Agent

考研数学多 Agent 智能学习系统

基于多 Agent 协作架构的考研数学智能学习系统。通过 4 个专业化 Agent 协同工作， 为考研学子提供从能力分析、智能出题、自动批改到学习规划的一站式学习体验。

┌─ [结论] 推理结论 ──────────────────────────────────────────────────────────┐
│ 知识点： 极限与连续 │
│ 难度： 简单 │
│ 题目： 求极限：lim_{x->0} sin(3x)/x
│ 答案： 3 │
│ 步骤： 利用重要极限 lim_{u->0} sin u / u = 1，变形后得 3 │
└─────────────────────────────────────────────────────────────────────────────┘
┌─ [题目] 数学题目 ──────────────────────────────────────────────────────────┐
│ 求极限： │
│ $$\lim_{x \to 0} \frac{\sin 3x}{x}$$
└─────────────────────────────────────────────────────────────────────────────┘
┌─ 解题步骤 ──────────────────────────────────────────────────────────────────┐
│ 步骤1：观察到当 x->0 时，分子 sin 3x -> 0，分母 x -> 0，是 0/0 型未定式。 │
│ 步骤2：利用重要极限：lim_{u->0} sin u / u = 1。 │
│ 步骤3：将原式变形：lim_{x->0} sin 3x / x = lim_{x->0} 3 * (sin 3x) / (3x)。 │
│ 步骤4：令 u = 3x，当 x->0 时 u->0， │
│ 所以原式 = 3 * lim_{u->0} sin u / u = 3 * 1 = 3。 │
└─────────────────────────────────────────────────────────────────────────────┘

安装使用

快速开始使用 MathExam AI Agent

1

克隆仓库

$ git clone https://github.com/happydayday/math-exam-agent.git
$ cd math-exam-agent
2

安装依赖

$ pip install -r requirements.txt
3

一键启动

$ python -m math_exam_agent start --mode mock

无需 API Key，开箱即用

常用命令

analyze
分析学习情况，生成掌握度报告
python -m math_exam_agent analyze --mode mock
practice
智能出题，针对薄弱点练习

个性化周计划生成
python -m math_exam_agent plan --mode mock
start
启动完整交互式学习会话
python -m math_exam_agent start --mode mock
